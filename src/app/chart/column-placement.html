This column chart demonstrates the dynamic sorting of data in real-time. The chart visualizes nitrogen fertilizer use per hectare of cropland, updating every few seconds to show sorted data from different datasets.

In this example, you can observe how the column chart updates dynamically with sorted data using the setData method. The data is sourced from multiple datasets, and the chart updates and sorts its data periodically to reflect the latest information.

Injecting Module

Chart component features are segregated into individual feature-wise modules. To use column feature, we need to inject ColumnService into the @NgModule.providers section.
